# Fuentes semillero

Fuentes públicas y comunidades para ampliar el corpus con información técnica y de producción verificable.

## Documentación técnica primaria

| Fuente | Plataforma | Contenido | URL de referencia |
|--------|-----------|-----------|-------------------|
| Amstrad CPC Firmware Guide | CPC | ROM calls, mapa de memoria, modos gráficos | `cpcwiki.eu` |
| The Complete Spectrum ROM Disassembly | Spectrum | ROM del 48K desensamblada y comentada | `skoolkid.github.io` |
| Commodore 64 Programmer's Reference Guide | C64 | Hardware, BASIC, mapa de memoria, VIC-II, SID | `c64-wiki.com` |
| MSX Technical Data Book | MSX | VDP, PSG, slots, BIOS | `msx.org` |
| Z80 CPU User Manual (Zilog) | CPC/Spectrum/MSX | Set de instrucciones, timing, interrupciones | `z80.info` |
| MOS 6502/6510 Reference | C64 | Set de instrucciones del procesador del C64 | `6502.org` |

## Wikis y bases de conocimiento comunitarias

| Fuente | Cobertura | Notas |
|--------|-----------|-------|
| **CPCWiki** (`cpcwiki.eu`) | CPC | La referencia más completa: hardware, software, demos, juegos |
| **World of Spectrum** (`worldofspectrum.org`) | Spectrum | Base de datos de juegos, revistas, documentación |
| **CSDb** (`csdb.dk`) | C64 | Base de datos de la demoscene C64 (código, técnicas) |
| **C64-Wiki** (`c64-wiki.com`) | C64 | Wiki técnica y de juegos |
| **MSX Resource Center** (`msx.org`) | MSX | Foros, wiki, base de datos de software |
| **MSX Assembly Page** (`map.grauw.nl`) | MSX | Referencia técnica orientada a programación ASM |
