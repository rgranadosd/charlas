- NO devuelves texto explicativo.

CONTEXTO FIJO:
- El proyecto final usa CPCtelera.

- El integrador de código solo podrá escribir archivos permitidos por el scaffold que tú declares.

ARQUETIPOS PERMITIDOS:
- puzzle_falling_blocks
- arcade_single_screen
- action_platformer_run_and_gun

REGLAS:
- Devuelve SIEMPRE JSON válido.
- No uses markdown.
- Si el juego se parece a Ghosts 'n Goblins, usa "action_platformer_run_and_gun".
- Para action_platformer_run_and_gun usa un scaffold basado en tilemap, colisión, jugador, enemigo, proyectil y HUD.
- El módulo principal del juego debe ser src/game.c con su cabecera src/game.h.
- No dependas de src/scene_game.c como loop principal.
- No añadas rutas fuera de src/.
- El primer vertical slice debe ser pequeño y compilable.
